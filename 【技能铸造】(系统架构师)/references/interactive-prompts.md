# 💬 Skill Creator 交互引导参考

## 1. 启动话术
> "你想做一个什么样的 Skill？简单来说，你希望只要**【输入】**什么，我就会**【输出】**什么？我会带你一步步把它做出来。"

## 2. 需求挖掘引导 (Phase 1)
- **问题**: "你希望我帮你做什么事情？"
- **选项**:
    - "处理文件 (比如 PDF、Excel、图片等)"
    - "帮我写东西 (比如文档、代码、报告)"
    - "连接某个服务 (比如发消息、查数据)"
    - "其他 (我来描述)"

## 3. 补充需求引导
- **问题**: "根据你的需求，我想到几个你可能也需要的："
- **选项**:
    - "[基于当前任务推测出的潜在需求]"
    - "暂时不需要，先做核心功能"

## 4. 期望标准引导
- **问题**: "你觉得这个 Skill 做得好，最重要的是什么？"
- **选项**:
    - "速度快"
    - "质量高/精准"
    - "操作简单"

## 5. 测试反馈引导
- **问题**: "测试结果怎么样？"
- **选项**:
    - "很好，完成了"
    - "有点问题，我说一下"
    - "完全不对，重新来"
